Forfatternet

Source filerne kan downloades her:

https://github.com/HenrikBirkholm/forfatternet

Forfatternet er en REST services, der kan omsætte et navn eller en liste af navne til en json struktur, hvor
navnet er delt op i et eller flere fornavne og et efternavn.

Servicen er implementeret i en jar fil, som kan downloades her:

https://github.com/HenrikBirkholm/forfatternet/blob/master/jar/forfatternet.jar

Services startes ved følgende kommando:

java -jar forfatternet.jar

Der er lavet en POSTMAN fil med tre eksempler på, hvordan servicen virker. POSTMAN filen kan importeres fra her:

https://github.com/HenrikBirkholm/forfatternet/blob/master/forfatternet.postman_collection.json
